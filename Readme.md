# 🤖 Automação Web com Selenium

Robô que automatiza login e navegação em sites usando Python e Selenium.

## O que o projeto faz

- Abre o navegador automaticamente
- Faz login em site de teste (SauceDemo)
- Localiza elementos por ID e CSS Selector
- Clica em botões automaticamente

## Como rodar

```bash
pip install selenium webdriver-manager
python automacaoweb.py
```

## Stack

- Python 3.14
- Selenium
- WebDriver Manager
